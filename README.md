# Ассистент для конвертации валют с реальным API

## Описание проекта

Веб-приложение на Streamlit, которое по текстовому запросу пользователя
выполняет конвертацию валют, используя **реальный публичный API** и
**LangChain**.

Пользователь пишет запрос обычным языком (например, на русском),
а приложение:

1. Анализирует запрос и определяет намерение (какую сумму, из какой валюты во что и на какую дату нужно перевести) с помощью LLM-цепочки.
2. Через агента с инструментом вызывает публичный API валют (Frankfurter API).
3. Возвращает пользователю понятное объяснение на русском:
   - исходная сумма и валюты
   - использованный курс
   - итоговая сумма
   - дата, на которую взят курс.

### Используемые компоненты LangChain

Проект демонстрирует работу нескольких ключевых компонентов LangChain:

1. **LLM (ChatOpenAI — модель `gpt-4o-mini`)**
2. **PromptTemplate** — шаблон промпта для определения намерения пользователя.
3. **Chains** — `LLMChain` для интерпретации запроса (`intent_chain`).
4. **Tools** — кастомный инструмент `convert_currency`, который ходит в реальный валютный API.
5. **Agent** — `STRUCTURED_CHAT_ZERO_SHOT_REACT_DESCRIPTION` агент, который:
   - интерпретирует нормализованное намерение и текст пользователя,
   - автоматически вызывает инструмент `convert_currency`,
   - формирует финальный ответ пользователю.

## Требования к окружению

- Python ≥ 3.10
- Установленный `pip`
- Аккаунт и API-ключ OpenAI (для модели gpt-4o-mini или другой совместимой модели)
- Рекомендуется использовать виртуальное окружение (`venv`, `conda` и т.п.)
